# homebrew-tap

Homebrew tap for [choolake](https://github.com/choolake) tools.

## Usage

```bash
brew tap choolake/tap
brew install keymaprd
```

## Formulae

| Formula | Description |
|---------|-------------|
| [keymaprd](Formula/keymaprd.rb) | Remap Logitech MX Master mouse buttons to keyboard shortcuts on macOS |
